Cardisan

Complete eCommerce project using the WooCommerce

Challenge: Our client decided to start an innovative business, however, before a full-fledged launch, he would like to check whether this idea will generate income. In case of a successful test launch, a sterling project must be developed. Our task was to develop an eCommerce service with unique functionality. The customer also wanted to be able to manage all content and products from a single Dashboard, as well as be able to import the product database in a few clicks to avoid the lengthy process of manually adding products.

Solutions: XPikTech team developed and launched MVP (minimum viable product). After a successful test run, we developed a complete eCommerce project using the WooCommerce WordPress plugin. We provided our customer with a Dashboard in which he can manage all content and goods. We have implemented a product import function and added it to the Dashboard.

Technologies: PHP, AJAX, WordPress, WooCommerce, MemberPress, Advanced Custom Fields, JavaScript, HTML/CSS.
