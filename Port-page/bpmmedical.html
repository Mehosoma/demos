BPM Medical

Design, draw, and develop a website for a private medical clinic

Domain Area: Medicine

Challenge: The task was to design, draw, and develop a website for a private medical clinic. The site must be in two languages, and the site must be able to make an appointment. All content should be managed from the admin panel.

Solutions: Our designer drew several versions of the design, the customer chose the one he liked best and we finalized it. We used Advanced Custom Fields for content management. For multilingualism WPML plugin was used. The ability to make an appointment was implemented using the Booked plugin.

Technologies: WordPress, WPML, PHP, JavaScript, HTML, CSS, Advanced Custom Fields, Booked, ContactForm 7
