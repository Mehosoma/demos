A.J. DIAMOND CLUB

The task was to fully develop a mobile application and a personal account on the website

Domain Area: Sport

Challenge: This time the task was to fully develop a mobile application and a personal account on the website for fans of the football team. The personal account must contain the contact details of all club members, as well as a chat between the members. The mobile application should provide users with the opportunity to confirm participation in the event, reserve a seat in the stadium, as well as create their own events and invite other club members.

Solutions: The user can become a member of the club only by subscription. To implement this feature, we used the MemberPress and BuddyPress plugins. To implement the ability to create events, confirm/decline participation in an event, we used the BuddyPress plugin and modified it to meet our needs. To speed up the development and integration of a personal account and a mobile application, we have developed our own REST API for WordPress using the WP REST API plugin. To manage content from the admin panel, we used the Advanced Custom Fields plugin. Thus, we used WordPress as the Back-End part for the mobile application. This greatly accelerated the development process.

Technologies: WordPress, Advanced Custom Fields, MemberPress, BuddyPress, WP REST API, Ionic Framework, Angular, JavaScript, PHP, HTML, CSS, jQuery
