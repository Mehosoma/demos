CivilPE Practice

The task was to redesign the existing eCommerce site, integrate the developed design with WordPress and WooCommerce

Domain Area: eLearning, eCommerce

Challenge: The task was to redesign the existing eCommerce site, integrate the developed design with WordPress and WooCommerce, integrate WooCommerce with moodle, and synchronize products with the customer's Amazon products.

Solutions: We developed a new design, integrated it with WordPress and WooCommerce. We used the Edwiser Bridge to integrate WooCommerce and Moodle. We have developed our own module for integrating WooCommerce and Amazon products. We also developed a unique theme. All sections of the site are independent and the customer has the ability to add and remove any section with one click of the mouse. We also developed a function that allows you to change the order of any sections on the site using drag and drop. All sections are available in order to create completely new pages from them without using programming. Thus, we have built a complete visual editor for this theme.

Technologies: WorkPress, WooCommerce, PHP, Moodle, Amazon, Advanced Custom Fields, JavaScript, HTML, CSS, ContactForm 7, jQuery
