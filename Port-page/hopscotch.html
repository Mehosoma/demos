We developed a custom theme, made it as flexible as possible and moved all content management to the admin panel.

Domain Area: Food Service

Challenge: The customer wanted to be able to regularly update the restaurant's menu, gallery, where one could see photos of the institution and a video tour. All content should be managed from the admin panel. The customer also wanted to develop another website for the hotel chain based on this theme.

Solutions:We offered the client a design that would be easy to change so that it can be used on the website of a hotel chain after developing a theme for the restaurant. We developed a custom theme, made it as flexible as possible and moved all content management to the admin panel. Then they installed this theme on the second site and the customer independently replaced all the content and successfully created another site using one theme.

Technologies: WordPress, PHP, JavaScript, HTML, CSS, Advanced Custom Fields, ContactForm 7, jQuery
