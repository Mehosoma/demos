Baseline Learning

The task was to develop a SaaS service for students

Domain Area: eLearning

Challenge: The task was to develop a SaaS service for students with which they can improve their numeracy, reading, and writing skills. It was necessary to develop a system that allows you to create courses, process responses, analyze the text, and show the user his errors. The service must have a personal account, as well as a teacher's account, which must have access to the results of his students.

Solutions: To implement the SaaS model, we used the Stripe payment system. We used the Amazon Comprehend OCR library for spelling analysis. We wrote our own text editor as well as our own algorithm that highlighted user errors. We also created our own quiz plugin to implement the student testing process. Using Strapi greatly accelerated the development process and we created this service in just a month.

Technologies: Angular 6, Node.js, Strapi, AWS, Amazon Comprehend, Stipe, MongoDB, JavaScript, HTML, CSS, jQuery
